# Richard's Credit Authority

This repository defines the personal authority layer for Richard Duane Kindler.  
It governs all credit issuance, loan approvals, disbursements, and ledger synchronization across the GitDigital ecosystem.

## Core Principles
- **Tax-first governance**
- **Audit-ready workflows**
- **Identity-verified authority**
- **Strict milestone enforcement**
- **Dual-founder safety**

- ## Quick Start

1. Clone the repo.
2. Run `node scripts/validate-authority.js`.
3. Simulate workflows:
   - `node scripts/run-workflow.js issue_credit`
   - `node scripts/run-workflow.js approve_loan`
4. (Optional) Run MkDocs locally:
   - `pip install mkdocs mkdocs-material`
   - `mkdocs serve`
